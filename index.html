Tipos de Hosting

Hosting Gratuito

  • Es un alojamiento web gratuito es un servicio que permite a los usuarios publicar sitios web, y en algunos casos tener servicio de correo electrónico o base de datos de manera gratuita.

Ventajas y Desventajas del Hosting Gratuito

  • Una de las ventajas es el hosting que te ofrecen un espacio gratuito donde poder subir la web.

  • Sus desventajas son que no tiene garantia, el soporte no existe,añaden publicidad, los hosting gratuitos no te ofrecen una copia de seguridad y respecto al software no podemos actualizar a las ultimas versiones de software

Hosting Compartido

  • Es el alojamiento compartido o alojamiento virtual, en inglés Virtual hosting, es una de las modalidades más utilizadas por las empresas dedicadas al negocio del alojamiento web.

Ventajas y desventajas del Hosting Compartido

  • Las ventajas que caracteriza a este host son el facil manejo, precio economico, son multidominios y gestionan el mantenimiento del servidor.
  • Sus deventajas es el recurso limitado, fallo de seguridad, software y hardware compartido, personalizacion en configuraciones del servidor y responsabilidad.

Hosting para revendedores

  • Es un tipo de alojamiento web mediante el cual un plan de alojamiento dentro de un servidor permite a una cuenta revender alojamiento dentro del mismo plan, permitiendo así crear subdominios y agregar dominios externos, creando subplanes o planes de alojamiento.

Ventajas y desventajas de Hosting para revendedores

  • Sus ventajas son precios bajos para comenzar un negocio en internet y tiene una capacidad de reventa que te permite revender recursos y poder comenzar poco a poco.
  • Sus deventajas son la cantidad de sitios a hospedar, espacio y transferencia, independencia de IPs, Independencia de recursos y sobreventa.

Hosting VPS

  • Hosting VPS o servidor virtual privado es un metodo de particionar un servidor fisico en varios servidores virtuales de tal forma que todo funcione como si estuvieras ejecutando una sola maquina.

Ventajas y desventajas de Hosting VPS

  • Sus ventajas son la privacidad, flexibilidad para que puedas aumentar RAM y CPU, diferentes IPs, precio dividido y seguridad.
  • Sus deventajas son el costo elevado, estudios del tema, seguridad que puede haber una mala configuracion, limitaciones para arojar paginas Web.

Hosting dedicado

  • Hosting dedicado o servidor de alejamiento es dedicado a un ordenador comprado o arrendado que se utiliza para prestar servicios dedicados y son relacionados con el alojamiento web y otros servicios en red.

Ventajas y desventajas de Hosting dedicado

  • Sus ventajas son exclusividad, administracion, seguridad, mantenimiento, versatilidad, sistema operativo.
  • Sus deventajas son el precio y conocimientos previos.

Hosting en la Nube

  • Hosting en la Nube o Cloud Hosting es un tipo de alojamiento web que utiliza multiples servidores diferentes para equilibrar la carga y maximizar el tiempo de actividad.

Ventajas y desventajas de Hosting en la Nube

  • Sus ventajas disponen de un ahorro de dinero, tiene mas disponibilidad, flexibilidad de los recursos y tiene un uso facil en el panel de control.
  • Sus deventajas son la importacia de un proveedor de confianza y una tiene una seguridad a manos de un proveedor que podria tener un problema tecnico.s
